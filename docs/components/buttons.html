Basic markup

Default

The button style can be used on <a> and <button> tags

<a href="#" class="btn btn-default">Default</a>

<button type="button" class="btn btn-default">Default</button>

The default rendering is a white button with a solid 2 pixel wide border. The full colour version of the buttons are generally use as primary CTA. There should only be one CTA per page.

To render a full colour button, you can add the .plein class.



Other modifiers

.no-background: Remove background color of a button. (<button> only)

.txt-blanc: Force text color to white. (<button> only)

.md-padding, .lg-padding: Add more horizontal padding

Button groups

Wrap <button> tags into a <div class="btn-group"> to make toggle button groups. The colour is set on each button using the same scheme as for individual buttons

<div class="btn-group">
  <button class="btn btn-default">toujours</button>
  <button class="btn btn-default">quelque fois</button>
  <button class="btn btn-default">jamais</button>
</div>
